# CISA Reporter

Retrive CVEs from CISA (https://www.cisa.gov/) and display them in a user-friendly format.

![](cisa_reporter.png)

## Example config
```
{
    "services": [
        "Artifactory",
        "Confluence",
        "GitLab",
        "Jira",
        "Metadefender",
        "Sourcegraph",
        "TeamCity",
        "Teleport"
    ]
}
```
